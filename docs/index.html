üí° Getting Started

To create your first script:

  1. Open SafeRS Lite and go to the Script Editor tab
  2. Click "üìÑ Load Template" to get the BaseScript template
  3. Modify the template to create your custom automation
  4. Use the automation modules (self.mouse, self.color, etc.) in your script
  5. Run your script using the execution controls

Need help? Check out the Simple Click Script example for a complete working script.

üîó Available Automation Modules

When writing BaseScripts, you have access to these automation modules:

  • self.mouse - Mouse automation (click_coordinates, move, click_area)
  • self.color - Color detection (find_color, wait_for_color, does_color_exist)
  • self.keyboard - Keyboard input (press_key, send_string, press_key_combination)
  • self.timing - Timing functions (wait, wait_random, wait_until)
  • self.inventory - Inventory management (click_slot, is_slot_empty, get_slot_info)
  • self.spellbook - Spellbook interaction (click_spell, get_spell_info)
  • self.hotkey - Hotkey management (register_hotkey, unregister_hotkey)

üìù Scan Areas

Pre-configured scan areas are automatically available in your scripts:

  • self.inventory - 28-slot inventory grid (4√ó7)
  • self.play_area - Main game interaction area
  • self.spellbook - Standard spellbook interface
  • self.bank - Bank interface area
  • self.chatbox - Game chat interface
  • self.minimap - Mini map area
  • self.health_orb - Health status orb
  • self.prayer_orb - Prayer status orb
  • self.run_orb - Run energy status orb
  • self.special_attack_orb - Special attack status orb
